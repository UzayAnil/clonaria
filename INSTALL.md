Dependencies
============
* cpython/pypy 2.7
* python-dev
* swig

Instructions
============
    # In a virtualenv or as root:
    git clone https://github.com/rettigs/clonaria.git
    cd clonaria
    pip install -r requirements.txt
    cd clonaria
    ./main.py

Notes
=====
main.py may be passed the -h or --help flags to view available command line options.
